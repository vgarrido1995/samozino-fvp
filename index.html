📖 Instrucciones de uso

⚠️ Importante: el menú «Cargar arquetipo» de la pestaña Datos solo sirve como ejemplo demostrativo (sprinter, saltador, halterófilo, etc.) para ver cómo luce un perfil F-v. NO lo selecciones si vas a introducir tus propios datos: sobrescribirá toda la sesión. Para tu evaluación real, déjalo en «— elegir —» y rellena los campos manualmente.

1. Atleta y sesión

  • Body mass (kg): masa corporal del atleta.
  • Hpo (m): distancia de empuje vertical, medida una sola vez por atleta.
    Con cinta métrica: distancia entre el trocánter mayor en posición baja del SJ (rodilla ~90°) y el trocánter en extensión completa de pie con tobillo en flexión plantar máxima. Típico 0.30 – 0.45 m.
  • α (°): ángulo de empuje. Deja 90 (vertical puro) salvo investigación específica.

2. Condiciones de carga

  • Realiza el SJ con varias cargas. Mínimo 2, recomendado 5–6.
  • Ejemplo: 0% BW, +20%, +40%, +60%, +80%, +100% BW de masa adicional.
  • Por cada condición, 3 intentos (se toma el mejor como Hmax).
  • Las alturas se introducen en metros (p.ej. 0.42, no 42).
  • Si tienes plataforma de contactos: Hmax = g · tv² / 8.

3. Cálculo

La app ajusta F = Fo + Sfv · V por mínimos cuadrados y calcula:

  • Fo, Vo = -Fo/Sfv, Pmax = Fo·Vo/4.
  • Sfv óptima según Samozino 2012 (depende de Pmax/kg, Hpo y α).
  • FVimb = Sfv / Sfv_opt expresado en % del óptimo.

4. Lectura de resultados

  • 🟢 Well balanced: FVimb dentro de ±10% del óptimo.
  • 🔴 Déficit de FUERZA: FVimb < 90% → entrenar fuerza.
  • 🟠 Déficit de VELOCIDAD: FVimb > 110% → entrenar velocidad/potencia.
  • r² < 0.95 → revisa la ejecución técnica.
  • CV > 5% entre los 3 intentos → ejecución inconsistente.

5. Errores comunes

  • ❌ Introducir altura en cm (30) en vez de metros (0.30).
  • ❌ Olvidar Hpo (queda en 0).
  • ⚠️ Solo 2 condiciones → resultado poco fiable.

Fórmulas clave

F = m_total · g · (Hmax / Hpo + 1)
V = √(g · Hmax / 2)
Vo = -Fo / Sfv
Pmax = Fo · Vo / 4
FVimb = Sfv / Sfv_opt

📚 Bases científicas

La aplicación implementa el método simplificado de Samozino para estimar el perfil Fuerza-Velocidad-Potencia en movimientos balísticos del miembro inferior, así como la determinación del perfil F-v óptimo y el desequilibrio (FVimb).

Qué aporta cada estudio

  • 2008 — Valida un método sin plataforma de fuerzas: solo masa, Hpo y Hmax.
  • 2010 — Marco teórico integrador del salto vertical.
  • 2012 — Deriva la Sfv óptima que maximiza el desplazamiento del CoM.
  • 2014 — Define FVimb y muestra que reducirlo mejora el rendimiento balístico.
  • 2016 — Guía práctica de interpretación y prescripción de entrenamiento.

Equipo de referencia

  • Pierre Samozino — Univ. Savoie Mont Blanc
  • Jean-Benoît Morin — Univ. Côte d'Azur

Limitaciones

  • Asume empuje rectilíneo vertical.
  • Sensible a buena medición de Hpo y Hmax.
  • Mínimo 2 condiciones; idealmente 5–6 para fiabilidad.
  • Validado en SJ (Squat Jump).

Esta app es una herramienta docente y de soporte. No sustituye juicio profesional.

⬇️ Descargas para usar la app sin internet

Si prefieres trabajar en tu propio computador (modo offline, sin depender de la web), tienes dos opciones según tu perfil técnico:

🪟

Ejecutable Windows (.exe)

Para usuarios — sin instalar nada

Descarga un único archivo .exe de ~80 MB y haz doble click. No necesitas Python, ni librerías, ni configuración. Funciona en Windows 10/11 (64 bits).

⬇️ Descargar FVP_Jump_Profile.exe
⚠️ Aviso de Windows SmartScreen

Como el ejecutable no está firmado digitalmente (firma de pago), la primera vez Windows puede mostrar un aviso azul: pulsa Más información → Ejecutar de todas formas. Es un comportamiento normal para apps personales / open source.

📦 Otras releases

Histórico completo en github.com/vgarrido1995/samozino-fvp-app/releases.

🐍

Código Python (para VS Code / IDE)

Para desarrolladores — modificable

Descarga el código fuente en Python (Tkinter). Lo puedes abrir con Visual Studio Code, PyCharm o cualquier editor, modificarlo y ejecutarlo en tu equipo.

⬇️ Descargar código (.zip) 🔗 Ver repositorio en GitHub
▶️ Cómo ejecutarlo
  1. Descomprime el ZIP.
  2. Doble click a run.bat (instala dependencias y arranca la app).

O manualmente desde terminal:

pip install -r requirements.txt
python fvp_app.py

Requiere Python 3.9+.
